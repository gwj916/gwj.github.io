Prisma实践

发表于2023-11-25|更新于2023-11-25|Node.js数据库Prisma
|阅读量:

安装

npm i prisma @types/node

创建prisma文件

schema.prisma

要连接数据库，您需要将 Prisma 架构中 datasource 块的 url 字段设置为数据库连接 URL：

generator client {
  provider = "prisma-client-js"
}

//: 操作指定数据库
datasource db {
  provider = "mysql"
  url      = env("DATABASE_URL")
}

//: 定义模型
model Post {
  id        Int      @id @default(autoincrement())
  createdAt DateTime @default(now())
  updatedAt DateTime @updatedAt
  title     String   @db.VarChar(255)
  content   String?
  published Boolean  @default(false)
  author    User     @relation(fields: [authorId], references: [id])
  authorId  Int
}

model Profile {
  id     Int     @id @default(autoincrement())
  bio    String?
  user   User    @relation(fields: [userId], references: [id])
  userId Int     @unique
}

model User {
  id      Int      @id @default(autoincrement())
  email   String   @unique
  name    String?
  posts   Post[]
  profile Profile?
}

配置数据库信息

.env

url 是通过 .env 中定义的环境变量设置的：

mysql://USER:PASSWORD@HOST:PORT/DATABASE

要将数据模型映射到数据库架构，您需要使用 prisma migrate CLI 命令：

npx prisma migrate dev --name init
  1. It creates a new SQL migration file for this migration
    它为此迁移创建新的 SQL 迁移文件
  2. It runs the SQL migration file against the database
    它针对数据库运行 SQL 迁移文件

使用 Prisma Client 编写您的第一个查询

index.ts

import { PrismaClient } from "@prisma/client";

const prisma = new PrismaClient();

async function main() {

//: 新增数据
  await prisma.user.create({
    data: {
      name: "Alice",
      email: "alice@prisma.io",
      posts: {
        create: { title: "Hello World" },
      },
      profile: {
        create: { bio: "I like turtles" },
      },
    },
  });

//: 查找所有数据，包含子集
  const allUsers = await prisma.user.findMany({
    include: {
      posts: true,
      profile: true,
    },
  });

  console.dir(allUsers, { depth: null });

//: 更新数据
  const update = await prisma.post.update({
    where: { id: 1 },
    data: { published: true },
  });
  console.log(update);

//: 删除数据
  const deleteResult = await prisma.user.delete({
    where: { email: '1@qq.com' }
  })
  console.log(deleteResult);
}

main()
  .catch(async (e) => {
    console.error(e);
    process.exit(1);
  })
  .finally(async () => {
    await prisma.$disconnect();
  });
  1. Import the PrismaClient constructor from the @prisma/client node module
    从 @prisma/client 节点模块导入 PrismaClient 构造函数
  2. Instantiate PrismaClient 实例化 PrismaClient
  3. Define an async function named main to send queries to the database
    定义一个名为 main 的 async 函数以向数据库发送查询
  4. Call the main function
    调用 main 函数
  5. Close the database connections when the script terminates
    脚本终止时关闭数据库连接

现在使用以下命令运行代码：

npx ts-node index.ts
