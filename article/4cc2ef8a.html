gulp

gulp是一个自动化构建工具，主要用来设定程序自动处理静态资源的工作。简单的说，gulp就是用来打包项目的

基本语法

导入gulp
const gulp = require('gulp');
文件系统读取
gulp.src('路径')
创建任务
gulp.task(任务名,处理函数)
文件输出目录
gulp.dest('路径')
热更新/监控
gulp.watch('路径',任务名称)
顺序任务队列
gulp.series(任务...)
并行任务队列
gulp.parallel(任务...)
删除任务
del()
管道函数
pipe(任务)

常用插件

gulp-cssmin–压缩css文件
npm install gulp-cssmin -D
gulp-autoprefixer–浏览器css前缀–配置加在package.json里
npm install gulp-autoprefixer -D
gulp-sass/scss/less–转css文件
npm install gulp-sass/scss/less -D
gulp–uglify–压缩js文件
npm install gulp-uglify -D
gulp–babel+–es6->es5转换
npm install gulp-babel  @babel/core @babel/preset-env -D
gulp-htmlmin–压缩html
npm install gulp-htmlmin -D

collapseWhitesp ace: true,//移除空格
removeEmptyAttributes: true,//表示移除空的属性
collapseBooleanAttributes: true,//移除布尔值
removeAttributeQuotes:true//移除双引号
minifyCSS: true,//压缩css
minifyJS: true,//压缩Js
removeStyleLinkTypeAttributes: true//移除style和link的type属性
removeScriptTypeAttributes: true,//移除script标签type属性
gulp-webserver–gulp服务器
npm install gulp-webserver -D

{--配置
    host:'127.0.0.1', 
    port:'8001',
    livereload: true,
    open:'./pages/index.html'
}
