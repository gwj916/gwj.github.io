typescript语法及使用

发表于2023-06-24|更新于2023-06-25|JavascriptJavascript框架Typescript
|阅读量:
数组
let arr: number[] = [1, 2, 3]
let arr2: Array<number> = [1, 2, 3]
函数可选参数
function printName(obj: { first: string; last?: string }) {
  // ...
}
接口
interface a{
    name: string
}
interface b extends a{
    sex:string
}
type Animal = {
  name: string
}
type Bear = Animal & { 
  honey: boolean 
}
类型断言
const myCanvas = document.getElementById("main_canvas") as HTMLCanvasElement;
const myCanvas = <HTMLCanvasElement>document.getElementById("main_canvas");
形参枚举
enum obj{a,b,c,d,e,f}
typeof 操作符-类型查询
function point(point:typeof p){}
implements 类 继承 接口
class son implements  father{} //方法里必传接口方法
类访问控制修饰符

public（默认） : 公有，可以在任何地方被访问。

protected : 受保护，可以被其自身以及其子类访问。

private : 私有，只能被其定义所在的类访问。
可见修饰符
readonly  //只能在构造函数进行再次赋值
泛型工具类型
Partial<type> 将所有属性设置为可选
Record<key|...,类型> 对象可选键，固定类型
Readonly<type> 将所有属性设置为只读
Pick<type,Keys> 将属性设置为传入的类型里的类型
