Apollo GraphOS

pollo GraphOS 是用于构建超级图的开发人员平台：组织数据和服务的统一网络，全部组成一个分布式 GraphQL API。

Apollo 为每个与 GraphQL 交互的开发人员提供工具——无论您是构建您的第一个 API、查询现有 API，还是将整个组织迁移到超级图上。

客户端使用一种称为 GraphQL 的强大语言查询您的超图，这使他们能够准确地检索他们需要的数据，而不会过度提取。超图的架构不会公开 - 客户端将查询发送到单个端点（路由器），无论它们需要什么数据。

基本用法

npm i @apollo/server graphql graphql-tag express @apollo/subgraph cors
  • graphql 是 GraphQL 的 JavaScript 参考实现
  • graphql-tag 是一个实用程序包，用于将 GraphQL 字符串解析为标准 GraphQL 抽象语法树 （AST）
  • @apollo/subgraph 是一个用于创建 GraphQL 微服务的实用程序包
  • @apollo/server 是一个符合规范的 GraphQL 服务器，它公开了一个 /graphql 端点
import { ApolloServer } from "@apollo/server";
import { expressMiddleware } from '@apollo/server/express4';
import gql from "graphql-tag";
import cors from 'cors';
import express from "express";
import { readFileSync } from "fs";
import { buildSubgraphSchema } from '@apollo/subgraph';

// 解析graphql文件
const typeDefs = gql(
  readFileSync("src/schema.graphql", {
    encoding: "utf-8",
  })
);

// 模拟数据
const books = [
  {
    title: "The Awakening",
    author: "Kate Chopin",  
  },
  {
    title: "City of Glass",
    author: "Paul Auster",
  },
];

// 定义实现方法
const resolvers = {

  // 定义查询方法
  Query: {
    books: () => books, 
  },
    
  // 定义变更方法
  Mutation:{
    createBook(_, { title, author }){
      const book = {
        title,
        author,
      };
      books.push(book); 
      return books;
    }
  }
};

// 定义服务，挂载schema的实现
const server = new ApolloServer({
  schema: buildSubgraphSchema({ typeDefs,resolvers  }),
});

// 定义expres示例
const app = express()

// 启动ApolloServer
await server.start();

// 使用express路由
app.use('/graphql', cors<cors.CorsRequest>(), express.json(), expressMiddleware(server));

// 监听5000端口
app.listen('5000', () => {
  console.log(`http://127.0.0.1:5000/graphql`);
});

打开浏览器UI界面查询

在浏览器中访问 http://localhost:4000 ，这将打开 Apollo Sandbox：

Apollo Sandbox UI 包括：

  • 用于编写和执行查询的操作面板（中间）

  • 用于查看查询结果的“响应”面板（右侧）

  • 用于架构浏览、搜索和设置的选项卡（左侧）

  • 用于连接到其他 GraphQL 服务器的 URL 栏（位于左上角）

查询图书列表
query ExampleQuery {
  books {
    title
    author
  }
}

新增图书
mutation ExampleMutation {
  createBook(title:"活着" , author: "余华") {
    title
    author
  }
}
